Documentation Page

This page lists the Bootstrap components and JavaScript features used in my one-page portfolio site.

Live Website URL

https://moulikakatragadda.github.io/MoulikaPortfolio/

Repository

https://github.com/Moulikakatragadda/MoulikaPortfolio.git

Bootstrap Components Used

Layout & Navigation

  • Navbar: responsive navigation with collapse functionality
  • Container: responsive fixed-width containers
  • Grid System: rows and columns for responsive layout
  • Flexbox Utilities: d-flex, align-items-center, justify-content-between, gap-*
  • ScrollSpy: highlights active section in the navbar while scrolling

Components

  • Cards: section blocks + skill/achievement boxes
  • Accordion: education modules by year
  • Modal: contact info popup + project details popup
  • Buttons: primary, outline, small button styles
  • Badges: tags for modules, projects, and highlights
  • Progress: skill bars + experience overview bars
  • Forms: contact form styling

Styling & Utilities

  • Typography: fw-bold, lead, headings for hierarchy
  • Spacing: py-5, mb-3, g-4, gap-*
  • Colors: light surfaces + purple accent theme (custom variables)
  • Rounded: rounded, rounded-3, rounded-circle
  • Images: img-fluid for responsive images
  • Height: h-100 to keep equal card heights

Icons

  • Bootstrap Icons: e.g. bi-envelope, bi-linkedin, bi-clipboard-check, bi-eye
  • Icon sizes: e.g. fs-5 for the LinkedIn icon

Dynamic Features & JavaScript

Bootstrap JavaScript Features

  • Navbar collapse: mobile menu toggle
  • Modal: contact popup + project details popup
  • Accordion: expand/collapse education years
  • ScrollSpy: updates active nav link while scrolling

Custom JavaScript

  • Scroll progress bar: shows reading progress at the top
  • Project filtering: buttons filter project cards by tag
  • Project modal content: modal pulls title/summary/bullets from button data attributes
  • Module search: filters module badges as you type
  • Skills slider: adjusts skill percentages based on the range slider value
  • Contact preview: live preview + 300-character limit
  • Theme toggle: button toggles dark/light theme and saves to localStorage

Responsive Design Features

Breakpoint Classes

  • col-lg-6, col-lg-7, col-lg-5: layout changes on large screens
  • col-md-4, col-md-6: multi-column layouts on tablet+
  • navbar-expand-lg: collapses navbar on smaller screens
  • g-* and gap-*: spacing that adapts well across sizes

Mobile Optimisation

  • Touch-friendly buttons and spacing
  • Responsive images using img-fluid / full-width section photos
  • Cards stack vertically on small screens
  • Sticky navbar for quick section navigation

Project Structure

Main Files

  • index.html: main portfolio page
  • documentation.html: this documentation page

External Dependencies

  • Bootstrap 5.3.3: CSS framework
  • Bootstrap Icons: icon library
  • Bootstrap JS bundle: interactive components (collapse, modal, accordion)
